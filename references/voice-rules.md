# Voice Rules & AI-ism Blacklist — Ghost-Writer

## Voice Calibration

Before writing, lock the voice in 4 dimensions:

| Dimension | Question | Examples |
|---|---|---|
| **Formal ↔ Casual** | "How do you talk to customers?" | Formal: "We provide comprehensive solutions." Casual: "We fix your shit." |
| **Serious ↔ Funny** | "Dry and direct, or do you drop jokes?" | Serious: "The data indicates..." Funny: "The data says you're screwed." |
| **Respectful ↔ Irreverent** | "Polite or 'call it like it is'?" | Respectful: "Some agencies struggle with..." Irreverent: "Most agencies suck at this." |
| **Matter-of-fact ↔ Enthusiastic** | "Just facts, or do you hype?" | M-o-f: "Revenue systems convert better." Enthusiastic: "Revenue systems are a GAME-CHANGER!" |

**Every sentence in the draft must match these 4 choices.**

---

## Voice Rules

1. **No corporate words.** Kill: leverage, utilize, synergy, ecosystem, holistic, cutting-edge, innovative, revolutionize, empower.
2. **Lead with the point.** First sentence carries the weight.
3. **Opinions have edges.** "This works because [reason]." Not "this is good."
4. **Numbers over adjectives.** "47% faster" not "huge improvement."
5. **Questions land harder.** "Most agencies don't know their CAC" > "You probably don't know your CAC."
6. **Short paragraphs.** 2-4 sentences. White space is readability.
7. **The reader is smart but busy.** Don't over-explain. Don't under-evidence.

### Tier 2 — Structural AI Patterns

| Pattern | Fix |
|---|---|
| "It's not X, it's Y." | Just state Y. |
| "The narrative is X. But the reality is Y." | Drop "the narrative is." Just say Y. |
| "This is the gap nobody's talking about." | State the gap directly. |
| "Here's what [they] won't tell you." | Use 1x per piece max. Better: just say the thing. |
| "The pattern is the same." | Cut it. |
| "Research shows that..." / "Studies have shown..." | Name the specific research or cut it. |
| "It is worth noting that..." | Delete. Just note it. |
| "Due to the fact that..." | "Because." |
| "In order to..." | "To." |
| "Has the ability to..." | "Can." |
| "A significant number of..." | Give the number or cut it. |
| "Many people believe..." | Who? Name them or cut it. |
| "Experts agree that..." | Which experts? Name them or cut it. |
| "In today's [adjective] [industry] landscape..." | Delete. Start with fact. |
| "As we navigate the [adjective] [noun] of..." | Delete. Start with the point. |
| "The [thing] has undergone a [adjective] shift..." | State what changed specifically. |

### Tier 3 — AI Rhythmic Patterns

- **Triplet fragments:** "Not X. Not Y. Z." → Rewrite as connected sentences.
- **Label-explain:** "Speed. X happens." → Integrate into flowing prose.
- **Stacked becauses:** Multiple "because" clauses in one sentence → Break into 2 sentences.
- **Multiplier escalation:** "10x... 100x..." → Give one specific number or delete.
- **Smug AI-vs-human punchlines:** "While humans struggle with X, AI handles it effortlessly." → Delete. Nobody likes this.

## AI-ism Self-Audit

Before shipping ANY draft:

1. Scan for Tier 1 words. Count hits. 0-2 = clean. 3-5 = suspicious. 6+ = purge.
2. Scan for Tier 2 structural patterns.
3. Scan for Tier 3 rhythmic patterns.
4. Check for em dashes. Count must be zero.
5. Read aloud. If any sentence sounds like it could appear in 100 other blog posts, rewrite it.
