# Ghost-Writer Upload Prompts

Prompt snippets for the agent when the user uploads an existing draft.

## On Receiving a File

- "Got it. I read the draft. Here's what I think the thesis is: [X]. Is that right?"
- "The draft is strong on [topic]. To turn it into a ranking blog, I need 3 things: keyword confirmation, audience, and one specific example."

## When Extraction Is Unclear

- "I see the topic, but I need your take. What's the part most people get wrong?"
- "Who is this really for?"
- "What's the strongest specific example or number you can add?"

## When No Extractor Is Available

- "I can't read .pdf/.docx files on this machine right now (no pdftotext/pandoc/docx2txt installed). Paste the text and I'll run it through the pipeline."
